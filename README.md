# Typescript Node Starter
>Personal starter project used for building Node.js apps in Typescript.

**Clone n' go**

```
git clone https://github.com/jaygraffagna/ts-node-starter.git
```

## Getting Started
1. Rename dir `mv ts-node-starter <project-name>`
2. Open dir `cd <project-name>`
3. Set remote origin
    ```
    git remote set-url origin <repo-url>
    ```
4. Install dependencies `npm install`
5. Update `package.json`
    - name
    - description
    - repository.url
    - keywords
    - author
    - bug.url
    - homepage
6. Run `npm start`
7. Code!
